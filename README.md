# rules-us-nv

Nevada tax and benefit rules encoded in Cosilico DSL.

## Structure

```
rules-us-nv/
├── statutes/       # Nevada Revised Statutes (NRS)
├── regulations/    # Nevada Administrative Code (NAC)
└── README.md
```

## Sources

- **Nevada Revised Statutes (NRS)**: https://www.leg.state.nv.us/NRS/
- **Nevada Administrative Code (NAC)**: https://www.leg.state.nv.us/NAC/

## Usage

This repository contains Cosilico DSL rules for Nevada state programs. Rules are organized by:

- `statutes/` - Rules derived from Nevada Revised Statutes
- `regulations/` - Rules derived from Nevada Administrative Code
